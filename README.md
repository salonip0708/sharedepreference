# SharedPreferences Flutter App

A simple Flutter application that demonstrates how to use **SharedPreferences** for local data storage. Users can enter text, save it permanently, load saved data anytime, and delete stored data easily.

## Features
- Save text data locally using SharedPreferences
- Load saved data instantly
- Delete stored data
- Clean and simple user interface
- Fast and lightweight local storage

## Technologies Used
- Flutter
- Dart
- SharedPreferences Package

## How It Works
1. Enter any text in the input field
2. Click **Save Data** to store the text locally
3. Click **Load Data** to retrieve saved text
4. Click **Delete Data** to remove stored data

## Project Structure
- `main.dart` → Main application file containing UI and logic
- SharedPreferences → Used for storing data locally

## Output
The app displays the saved text on screen after loading data.

## Use Cases
- Save user preferences
- Store login status
- Save small app settings
- Remember notes or text inputs

## Conclusion
This project is a beginner-friendly example to learn local storage in Flutter using SharedPreferences.
